NAME:
   maze - Generate and solve mazes

USAGE:
   maze [global options] [command [command options]]

VERSION:
   dev

COMMANDS:
   generate  Generate a maze using specified algorithm
   solve     Solve a maze using specified algorithm
   help, h   Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
